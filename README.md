# 📌 EduMate (에듀메이트)

AI 기반 강의 이해도 분석 및 학습 지원 시스템

---

## 📖 프로젝트 소개
강의 영상, 강의 자료, 퀴즈 및 과제 데이터를 분석하여  
학생의 이해도를 파악하고 학습을 지원하는 AI 시스템

---

## 🚀 주요 기능
- 강의 자료 및 영상 분석
- 강의 내용 자동 요약 생성
- 사용자 수준 기반 예상 문제 생성
- 오답 기반 학습 피드백 제공

---

## 🛠️ 기술 스택
- AI: PyTorch, Transformers
- Backend: Django
- Frontend: HTML, CSS, JavaScript

---
